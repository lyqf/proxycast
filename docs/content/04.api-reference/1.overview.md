# API 概览

::alert{type="info"}
本章节面向进阶用户与开发者。普通创作者可直接在应用内使用，无需 API 接入。
::

当你需要把 ProxyCast 接入脚本、自动化流程或第三方工具时，可使用本地 API。

## 常见端点类型

### 通用对话端点

用于文本生成、对话续写等任务。

### 模型与管理端点

用于读取模型列表、状态信息和部分管理能力。

### 扩展端点

用于特定平台或集成场景。

## 认证方式

- `Authorization: Bearer <api-key>`
- 或使用兼容格式的密钥头

请确保 API Key 仅在可信环境使用。

## 默认地址

默认本地地址：`http://127.0.0.1:8999`

通常建议保持本地监听，不对公网暴露。

## 错误排查建议

- `401`：密钥错误或请求头格式错误
- `404`：端点路径错误
- `429`：请求频率过高
- `5xx`：服务端异常或上游波动
