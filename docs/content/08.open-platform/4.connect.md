# 开放平台 - Connect

::alert{type="info"}
本页面向生态合作方与平台接入方。普通创作者可跳过。
::

Connect 通过 Deep Link 提供“一键配置”能力，帮助外部平台将配置快速带入 ProxyCast。

## 核心价值

- 用户：减少手动配置步骤
- 服务提供方：降低接入门槛
- 平台：提升配置成功率

## 一键配置流程

1. 用户在外部平台点击一键配置
2. 浏览器打开 `proxycast://` 链接
3. ProxyCast 弹出确认
4. 用户确认后完成导入

## 基础协议

```text
proxycast://connect?relay={relay_id}&key={api_key}&name={key_name}&ref={ref_code}
```

| 参数 | 必填 | 说明 |
|------|------|------|
| `relay` | ✅ | 服务方唯一标识 |
| `key` | ✅ | API Key |
| `name` | ❌ | 显示名称 |
| `ref` | ❌ | 推广或渠道标记 |
