# 开放平台 - 插件开发指南

::alert{type="info"}
本页面向开发者。若你不开发插件，可跳过。
::

本文档说明插件类型、打包方式和开发建议，帮助你把能力稳定接入 ProxyCast。

## 插件类型

### 脚本插件

基于 JavaScript/TypeScript，通过 Hook 扩展行为。

### 二进制插件

通过独立可执行文件提供系统级能力，适合重计算或本地工具集成。

## 基础结构

插件以 ZIP 分发，至少包含：

- `plugin.json`：元数据与入口定义
- 可选配置文件与资源文件

## 开发建议

1. 先做最小可用版本
2. 明确输入输出协议
3. 做异常与超时处理
4. 提供清晰的版本兼容说明

## 发布建议

1. 版本号语义化管理
2. 发布前做跨平台验证
3. 提供回滚策略与变更日志
